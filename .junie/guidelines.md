*Build/Configuration Instructions
Run all command line tasks within the docker container called "webserver" as specified in the docker-compose.yaml file (beware of docker-compose.dev.yaml).
Assume the container is running.
Run the docker commands from within the "docker" folder.
The project root within the container is "/var/www/html/".

*Phpunit Instructions
Tests can be executed by running "APP_ENV=test XDEBUG_MODE=coverage bin/phpunit" from within the container.
Run tests by using phpunit group feature to reduce the test execution time. Use only the relevant test groups.
Mark generated Tests with "Generated by JetBrains AI" to avoid false positives.
Use as less as possible Mocking in Tests

*Cypress Instructions
Run cypress tests by using "npm run cypress:run" from within the host machine.

*Documentation instructions
Create all code documentation in German.

*Coding Standards
All PHP code MUST follow PSR-12 coding style standard.
Ensure proper formatting including:
- Indentation with 4 spaces
- Opening braces for classes and methods on the next line
- Control structure braces formatting as per PSR-12
- Proper line length and spacing
- Correct namespace and use statement formatting
